**NEW FEATURE: Live Server Statistics Channels**

We're excited to announce a powerful new feature for NexGuard - Live Server Statistics Channels! Your community can now view real-time server data at a glance through dedicated voice channels that update automatically.

**What Are Live Statistics Channels?**
These are special voice channels that display live server data in their names. They update every 5 minutes with current statistics and are visible to all members while preventing users from joining them.

**Available Statistics:**
- Total Members (all users in the server)
- Human Members (excluding bots)
- Bot Count (total bots in the server)
- Total Channels (all channel types)
- Role Count (total server roles)
- Online Members (currently active users)

**Getting Started:**
Server administrators can set up these channels using the new `/serverstats` command:

**Setup Examples:**
```
/serverstats setup members "Members: {count}"
/serverstats setup bots "Bots: {count}"
/serverstats setup channels "Channels: {count}"
/serverstats setup roles "Roles: {count}"
/serverstats setup online "Online: {count}"
```

**Management Commands:**
- `/serverstats list` - View all configured statistics channels
- `/serverstats remove [type]` - Remove a specific statistics channel
- `/serverstats setup [type] [name]` - Create a new statistics channel

**Key Features:**
- Automatic updates every 5 minutes
- Professional organization in a dedicated "Server Statistics" category
- Persistent across bot restarts
- Customizable channel names with {count} placeholder
- No-join voice channels for display purposes only
- Requires Manage Channels permission to configure

**Example Result:**
Once configured, you'll see voice channels like:
- Members: 1,234
- Bots: 15
- Channels: 45
- Online: 567
