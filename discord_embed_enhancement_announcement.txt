**NexGuard Bot Update: Enhanced Embeds & Ticket Management**

We've rolled out major improvements to both embed creation and ticket management systems!

**What's New:**

**Universal Newline Support**
- The /embed command now supports \n characters for multi-line text
- Works in all embed fields: title, description, footer, and author
- Multiple newline formats supported: \n, \\n, \r\n, \r
- Universal support across both ticket embeds and regular embeds

**New Ticket Management Commands**
- **/close** - Staff can instantly close tickets with automatic transcript generation
- **/close-request** - Users can request ticket closure with staff approval buttons
- **/add** - Add additional users to existing tickets with proper permission management

**Enhanced Reliability**
- Resolved interaction timeout errors that were causing command failures
- Improved error handling across all embed and ticket systems

**How to Use:**

**Multi-line Embeds:**
```
/embed title:"Welcome Message" description:"Line 1\nLine 2\n\nLine 4 after blank line"
/embed title:"Multi-line Title\nSubtitle" footer:"Footer line 1\nFooter line 2"
```

**Ticket Management:**
```
/close - Close current ticket with transcript
/close-request - Request ticket closure (users)
/add @user - Add user to current ticket
```
