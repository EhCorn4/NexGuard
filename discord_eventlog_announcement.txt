## ⚙️ **Advanced Configuration**

Configure specialized logging channels for different event types:
- `/eventlog set member #member-logs` - Member activity tracking
- `/eventlog set message #message-logs` - Message monitoring
- `/eventlog set voice #voice-logs` - Voice activity tracking
- `/eventlog set moderation #mod-logs` - Moderation actions
- `/eventlog set channel #server-logs` - Server changes
- `/eventlog set role #role-logs` - Role management
- `/eventlog set general #general-logs` - Everything else

**Test Your Setup:**
- `/logtest member` - Test member event logging
- `/logtest message` - Test message event logging
- `/eventlog view` - View all current logging settings
