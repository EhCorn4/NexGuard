# We have tickets!!!

🎫 **Professional Ticket System Now Available!**

Your server now has access to a comprehensive ticket support system that rivals dedicated ticket bots like TicketsBot.net! Here's everything you need to know:

## 🚀 What This Does
- **Interactive Panel Buttons** - Deploy beautiful embed panels anywhere in your server
- **Smart Channel Management** - Tickets create private channels with format: `{panel}-{username}` (e.g., "support-john", "billing-sarah")  
- **Professional Controls** - Close 🔒 and Claim 🙌 buttons for staff management
- **Automatic Transcripts** - Complete conversation history sent to all participants when tickets close
- **Custom Embeds** - Separate customization for panel messages vs ticket channel messages
- **Role/User Pings** - Notify your support team automatically when tickets open
- **Placeholder Variables** - Dynamic content with `{user.mention}`, `{user.name}`, `{guild.name}`, and line breaks

## 🛠️ How It Works
1. **Create a panel** with custom embeds and settings
2. **Deploy the panel** to any channel for users to see
3. **Users click the button** to instantly create a private ticket channel
4. **Staff claim tickets** and provide support
5. **Close tickets** with automatic transcripts and channel cleanup

## 📋 Quick Setup Commands

**Create Your First Panel:**
```
/ticket-panel action:create panel_id:support title:"General Support" 
  ticket_embed_description:"Hello {user.mention}!\n\nWelcome to {guild.name} support.\nPlease describe your issue and we'll help you."
```

**Deploy to Any Channel:**
```
/ticket-panel action:deploy panel_id:support channel:#general
```

**Advanced Features:**
- Set ticket categories: `category:#ticket-category`
- Ping support roles: `roles:@Support Team @Moderators`
- Custom panel embeds with headers, titles, descriptions
- Custom ticket channel embeds with placeholder variables

## 🎨 Customization Options
- **Panel Embeds**: What users see before opening tickets
- **Ticket Embeds**: Welcome message in ticket channels  
- **Role Pings**: Notify specific teams (only appears in tickets, not panels)
- **Categories**: Organize tickets in Discord categories
- **Deletion Timers**: Custom channel deletion delays (0-300 seconds)

## 📚 All Available Commands
- `/ticket-panel action:create` - Create new panels
- `/ticket-panel action:edit` - Modify existing panels  
- `/ticket-panel action:deploy` - Deploy panels to channels
- `/ticket-panel action:list` - View all your panels
- `/ticket-panel action:delete` - Remove panels

## 💡 Pro Tips
- Create multiple panels for different purposes (Support, Billing, Reports, etc.)
- Use placeholder variables like `{user.mention}` and `\n` for line breaks
- Set up categories to keep tickets organized
- Configure role pings to notify the right teams instantly
